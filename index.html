PIPE*.gml = 主要管線，點擊會顯示管徑、材質、管線種類、深度。
HYDRANT*.gml、HYDRANTL*.gml、VALVE*.gml、PIPECOVER*.gml、PIPEHAT*.gml = 附加圖資，只顯示，不跳資訊；PIPECOVER 以「=」符號顯示，PIPEHAT 以「ㄇ」字型顯示。

ℹ️ 使用提醒
1. 先把 Apps Script 後端部署成 Web App。
2. 把 Web App 網址貼到上方後按 💾 儲存設定。
3. Google Drive 直傳需另外建立 Web application OAuth Client ID，並把本站網址加入 Authorized JavaScript origins。
4. 填好 OAuth Client ID 後，先按 🧪 登入 / 測試 Drive 完成 Google 授權。
5. 按下 💾 儲存丈量紀錄 時，資料與照片只會先暫存在本機。
6. 需由使用者手動按 ☁️ 同步，才會同步丈量資料並把照片上傳到 Google Drive。
7. 圖片會用 photo_01、photo_02... 形式覆蓋重存，避免重複照片。
8. 管線圖資可把 PIPE、HYDRANT、HYDRANTL、VALVE、PIPECOVER、PIPEHAT 等 GML 放在同一個 Drive 資料夾；系統會依檔名自動分類，按 🔄 更新圖資 後會重新下載並更新本機快取。
⚠️ 刪除確認
只會刪除本機中的此筆案件（丈量清單、丈量地圖）。後端與 Google Drive 不會被刪除，之後按同步也不會送出這筆本機刪除資訊。
